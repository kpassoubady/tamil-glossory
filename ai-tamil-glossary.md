# தமிழ் AI கலைச்சொற்கள் — பதிப்பு 3 (Tamil AI Glossary — Version 3)

**குறிப்பு:** இந்தப் பதிப்பு கீழ்க்கண்ட நெறிமுறைகளைப் பின்பற்றுகிறது:

1. **சொல்லாய்வு குழுவின் கலைச்சொற்களுக்கு முன்னுரிமை** — அவை இயல்பாகவும், பயன்படுத்தத்தக்கதாகவும் இருக்கும்போது.
2. **விதிவிலக்குகள்** — Asynchronous/Synchronous போன்ற ஏற்கனவே நிறுவப்பட்ட சொற்களுக்கு (ஒத்தியங்கா/ஒத்தியங்கு).
3. **ஒற்றுச் சேர்ந்த ஒற்றைச் சொல் வடிவம்** (compound words joined: நரவலை, சொல்துண்டு, தரவுக்கணம்).
4. **a + b + c சொல் வேர் விளக்கம்** சாத்தியமான இடங்களில் சேர்க்கப்பட்டுள்ளது.
5. **வடசொற்கள்** இயன்றவரை நீக்கப்பட்டுள்ளன (பாரபட்சம் → சார்பு; அங்கீகாரம் → அறிதல்; சமிக்ஞை → குறிப்பு; ஆரம்பம் → தொடக்கம்; நிஜம் → உண்மை).
6. **ஒருமை வடிவம்**: கலைச்சொற்கள் இயன்றவரை ஒருமை வடிவில் தரப்படுகின்றன; தேவையான இடங்களில் தமிழின் இயல்பான பன்மை விகுதி (-கள்) சேர்த்துப் பயன்படுத்தலாம்.

| Autonomous Workflow | தன்னாட்சிப் பணிப்பாய்வு | தன்னியக்கப் பணித்தொடர் | மனிதத் தலையீடின்றி, பல படிநிலைகளைக் கொண்ட சிக்கலான பணிகளை AI தாமாகவே திட்டமிட்டுச் செயல்படுத்தும் தொடர்முறை. |
| Autoregressive Model | தன்னிழைவு மாதிரி | முன்னிலையடிப்படை மாதிரி | தன் (self) + இழைவு (regression/relation) + மாதிரி (model). முந்தைய சொற்களை வைத்து அடுத்த சொல்லைக் கணிக்கும் மாதிரி (GPT போன்ற LLM-கள் இதன் அடிப்படையில் இயங்குகின்றன). |
| Backpropagation (of error) | பின்பிழைப்பாய்ச்சம் | பின்னோக்கிய பிழைப்பாய்ச்சம் | பின் (back) + பிழை (error) + பாய்ச்சம் (propagation). நரவலையில் பிழைகளைக் கணக்கிட்டு, எடைகளைச் சரிசெய்யப் பின்னோக்கிச் செல்லும் கணித முறை. [^1] |
| Backpropagation Neural Networks | பின்பிழைப்பாய்ச்ச நரவலை | — | பின்பிழைப்பாய்ச்ச முறையைப் பயன்படுத்திப் பயிற்றுவிக்கப்படும் நரவலைக் கட்டமைப்பு. [^1] |
| Batch Size | தொகுதி அளவு | — | ஒரு பயிற்சிச் சுழற்சியில் ஒரே நேரத்தில் பயன்படுத்தும் எடுத்துக்காட்டுகளின் எண்ணிக்கை. |
| Beam Search | கதிர்த்தேடல் | — | பல சாத்தியமான உரைத் தொடர்களை ஒருசேர வைத்துச் சிறந்ததைத் தேர்வு செய்யும் தேடல் முறை. |
| Beam Width | கதிரகலம் | தேடல் அகலம் | கதிர் (beam) + அகலம் (width). கதிர்த்தேடலில் ஒரே நேரத்தில் நினைவில் கொள்ளும் சிறந்த தொடர்களின் எண்ணிக்கை. |
| Benchmark | அளவுகோல் | தரப்படுத்தல் | வெவ்வேறு AI மாதிரிகளின் செயல்திறனை ஒப்பிட்டு மதிப்பிடப் பயன்படுத்தப்படும் தரப்படுத்தப்பட்ட சோதனைகள் மற்றும் தரவுக்கணங்கள் (எ.கா: MMLU, HumanEval). |
| Bias | சாய்வு | சார்பு | பால், இனம் போன்றவற்றின் அடிப்படையில் AI வெளியீட்டில் ஏற்படும் ஒருதலைச் சார்பு; பயிற்சித் தரவுக் குறைபாட்டால் உருவாவது. |
| Bias-Variance Tradeoff | சாய்வு-மாறுபாட்டு ஈடு | — | சாய்வு (bias) + மாறுபாடு (variance) + ஈடு (tradeoff). மாதிரியின் சாய்வையும் மாறுபாட்டையும் சமன்படுத்தும் பயிற்சி நெறிமுறை — மிகச் சாய்வாக இருந்தால் வழுச் சேர்க்கும், மிக மாறுபாடாக இருந்தால் மிகைப்பொருத்தம் ஏற்படும். |
| Blockchain | கட்டச்சங்கிலி | — | கட்டம் (block) + சங்கிலி (chain). தரவுகளை மாற்ற முடியாதபடி சங்கிலித் தொடராகத் தொகுத்துச் சேமிக்கும் பரவலாக்கப்பட்ட தரவுத்தளத் தொழில்நுட்பம். |
| Callback | கூவிளி | திருப்பி அழைப்பு | கூவு (call) + விளி (summon). ஒரு செயல் முடிந்த பிறகு தானாகவே இயக்கப்படும்படி முன்கூட்டியே அனுப்பப்படும் சார்பு. [^1] |
| Catastrophic Forgetting | பேரழிவு மறதி | — | பேரழிவு (catastrophic) + மறதி (forgetting). புதிய தரவில் நுண்திருத்தம் செய்யும்போது மாதிரி பழைய அறிவை இழக்கும் நிலை. |
| Causal Language Model | காரணிய மொழி மாதிரி | — | இடமிருந்து வலமாக அடுத்த சொல்லைக் கணிக்கும் மாதிரி (GPT); MLM-க்கு எதிர்மாறானது. |
| Chain of Thought (CoT) | சிந்தனைச் சங்கிலி முறை | படிநிலைச் சிந்தனை | சிந்தனை (thought) + சங்கிலி (chain). AI மாதிரியானது விடையை ஒரே அடியாகக் கூறாமல், மனிதனைப் போலப் படிப்படியாகக் காரணம், காரியங்களை விளக்கி விடை காணும் முறை. |
| ChatGPT | சாட்ஜிபிடி | சேட்ஜிபிடி | ஓபன்ஏஐ (OpenAI) நிறுவனம் உருவாக்கிய, இயல்பான மொழியில் உரையாடும் பெருமொழி மாதிரி (LLM). |
| Chunking | உரைத்துண்டாக்கம் | பகுதிப்பகுப்பு | உரை (text) + துண்டு (chunk) + ஆக்கம் (process). நீண்ட ஆவணத்தை AI புரிந்துகொள்ளச் சிறிய பகுதிகளாக உடைக்கும் செயல்முறை. |
| Circuit (Mech Interp) | சுற்றமைப்பு | இயக்கச் சுற்று | இயக்கமுறை விளக்கத்திறன் (Mechanistic Interpretability) ஆய்வில், ஒரு குறிப்பிட்ட செயலை நிறைவேற்ற நரவலையில் ஒன்றிணைந்து செயல்படும் நரம்புகள் மற்றும் இணைப்புகளின் தொகுதி. |
| Classification | வகைப்பாடு | — | தரவை முன்கூட்டியே தீர்மானிக்கப்பட்ட வகைகளில் ஒன்றில் சேர்த்தல். |
| Cloud Computing | முகில் கணிமை | மேகக் கணிமை | முகில் (cloud) + கணிமை (computing). இணையத்தின் வழியே சேவையகங்கள், தரவுத்தளங்கள் மற்றும் மென்பொருள்களைப் பயன்படுத்தும் தொழில்நுட்பம். [^1] |
| Clustering | கொத்தாக்கம் | — | கொத்து (cluster) + ஆக்கம் (process). ஒத்த பண்புகளைக் கொண்ட தரவுகளைக் குழுக்களாகப் பிரித்தல். |
| Code | நிரல் | குறிமுறை | கணினிக்குக் கட்டளைகளை வழங்கப் பயன்படுத்தப்படும் நிரலாக்க மொழித் தொடர்கள். |
| Compress | சுருக்கம் | தரவுச் சுருக்கம் | தரவின் அளவைக் குறைத்து, சேமிப்பக இடத்தையோ அலைவரிசையையோ மிச்சப்படுத்தும் செயல்முறை. [^1] |
| Computational Language Model | கணியமொழியொப்பாக்கம் | கணிய மொழி மாதிரி | கணியம் (computational) + மொழி (language) + ஒப்பாக்கம் (modeling). மொழியின் கட்டமைப்பைக் கணக்கீட்டு முறையில் வடிவமைத்துச் செயல்படும் மாதிரி. [^1] |
| Connectionism | குறியியக்கியம் | இணைப்புக்கொள்கை | குறி (sign/symbol) + இயக்கியம் (-ism / theory). மூளையின் நரம்பணு வலைப்பின்னலைப் போலக் கணினி அமைப்புகளையும் ஒன்றோடொன்று இணைத்துச் செயல்பட வைக்கும் அணுகுமுறை. [^1] |
| Constitutional AI | அடிப்படை விதிச் செய்யறிவு | நெறிமுறை அடிப்படைச் செய்யறிவு | AI மாதிரிக்கு ஒரு "அடிப்படை விதி" அல்லது நெறிமுறை வழிகாட்டுதலை அளித்து, அதன் வெளியீடுகள் அந்த விதிகளுக்கு ஏற்ப இருக்கப் பயிற்றுவிக்கும் முறை (Anthropic நிறுவனத்தின் கட்டமைப்பு). |
| Constitutional Classifier | அடிப்படை விதி வகைப்படுத்தி | — | Constitutional AI அமைப்பில், ஒரு உள்ளீடு அல்லது வெளியீடு அமைப்பின் அடிப்படை விதிகளை மீறுகிறதா என்று வகைப்படுத்தும் துணை மாதிரி. |
| Context Engineering | சூழல் வடிவமைப்பியல் | — | சூழல் (context) + வடிவமைப்பு (engineering) + இயல் (-ology). AI மாதிரிக்குச் சரியான சூழலை — ஆவணங்கள், கருவிகள், அறிவுறுத்தல்கள், உரையாடல் வரலாறு — முறையாக வடிவமைத்து வழங்கும் துறை. |
| Context Window | சூழல் சாளரம் | உள்ளீட்டு வரம்பு / சூழல் நீளம் | ஒரு AI மாதிரி ஒரே நேரத்தில் படித்து நினைவில் கொள்ளக்கூடிய ஒட்டுமொத்த சொல்துண்டுகளின் (Tokens) அதிகபட்ச அளவு. |
| Contextual Embedding | சூழல் உட்பொதிவு | — | சூழல் (context) + உட்பொதிவு (embedding). வாக்கியச் சூழலைப் பொறுத்து மாறும் சொல் திசையன்; நிலையான உட்பொதிவுகளுக்கு (Word2Vec) மாறாக BERT, GPT பயன்படுத்துவது. |
| Contrastive Learning | எதிர்வைப்புக் கற்றல் | ஒப்பீட்டுக் கற்றல் | எதிர்வைப்பு (contrast) + கற்றல் (learning). ஒத்த இணைகளை நெருக்கி, மாறுபட்டவற்றை விலக்கிக் கற்கும் முறை (CLIP). |
| Convolutional Neural Network (CNN) | சுருள் நரவலை | முறுக்கு நரவலை | சுருள் (convolution) + நரவலை (neural network). படங்களையும் வடிவப் பகுப்பாய்வையும் சிறப்பாகக் கையாளும் ஆழ்கற்றல் கட்டமைப்பு; படத்தின் சிறு பகுதிகளில் வடிவங்களைக் கண்டறிய சுருள் வடிவ வடிப்பான்களைப் பயன்படுத்துகிறது. |

| Culturally-aware AI | பண்பாட்டுணர்வுச் செய்யறிவு | பண்பாட்டுச் சார்பு செய்யறிவு | பண்பாடு (culture) + உணர்வு (awareness) + செய்யறிவு. ஒரு குறிப்பிட்ட பண்பாட்டின் மரபுகள், மொழி, மதிப்புகள் மற்றும் சூழமைவை உணர்ந்து செயல்படும் AI மாதிரி. |
| Data Augmentation | தரவு விரிவாக்கம் | — | ஏற்கனவே உள்ள பயிற்சித் தரவுகளிலிருந்து புதிய, மாறுபட்ட எடுத்துக்காட்டுகளைச் செயற்கையாக உருவாக்கிப் பயிற்சித் தரவின் அளவை அதிகரிக்கும் நுட்பம். |
| Data Centre | தரவு நடுவம் | தரவு மையம் | தரவு (data) + நடுவம் (center). கணினி அமைப்புகளையும், தொடர்புடைய தரவுச் சேமிப்பகங்களையும் ஒருங்கே மையப்படுத்தி நிர்வகிக்கும் கட்டமைப்பு. [^1] |
| Data Modeling | தரவு மாதிரியாக்கம் | — | உண்மை உலகச் சிக்கலையோ செயல்முறையையோ கணினி புரிந்துகொள்ளக்கூடிய தரவுக் கட்டமைப்பாக மாற்றும் முறை. |
| Data Points | தரவுப்புள்ளிகள் | — | தரவு (data) + புள்ளி (point). ஒரு தரவுக்கணத்தில் உள்ள தனிப்பட்ட அளவீடுகள் அல்லது அவதானிப்புகள்; AI மாதிரிகள் இவற்றிலிருந்து கற்றுக் கொள்கின்றன. |
| Dataset | தரவுக்கணம் | — | தரவு (data) + கணம் (set). இயந்திரக் கற்றலுக்குப் பயன்படும் தரவுத் தொகுப்பு. |
| Debugging | பிழைத்திருத்தம் | — | பிழை (bug/error) + திருத்தம் (correction). மென்பொருள் நிரலில் உள்ள தவறுகளைக் கண்டறிந்து சரிசெய்தல். |
| Decoder | குறிநீக்கி | மீள்வடிவாக்கி | குறி (code) + நீக்கி (remover). குறியாக்கப்பட்ட தரவிலிருந்து மனிதர்கள் அல்லது கணினி புரிந்துகொள்ளும் வெளியீட்டை உருவாக்கும் பகுதி. |
| Deductive Reasoning | விதி ஏரணம் | பகுப்பு ஏரணம் | விதி (rule) + ஏரணம் (logic). பொது விதிகளிலிருந்து குறிப்பிட்ட முடிவை எடுக்கும் ஏரண முறை (எ.கா: அனைவரும் இறப்பர் → சாக்ரடீஸ் இறப்பார்). |
| Deep Learning | ஆழ்கற்றல் | ஆழ்திறன் கற்றல் | ஆழ் (deep) + கற்றல் (learning). பல அடுக்கு நரவலைகளைப் பயன்படுத்திப் பெருமளவு தரவுகளிலிருந்து சிக்கலான அமைப்புகளைக் கற்றுக் கொள்ளும் முறை. |
| DeepFake | ஆழ்போலி | ஆழ்பொய் | ஆழ் (deep) + போலி (fake). ஆழ்கற்றல் நுட்பத்தைப் பயன்படுத்தி உருவாக்கப்படும் போலி காணொளி, படம் அல்லது ஒலி; உண்மையிலிருந்து வேறுபடுத்திக் காண்பது கடினம். [^1] |
| DeepSeek (company) | ஆழ்தேடி | ஆழ்தேடல் | ஆழ் (deep) + தேடி (seeker). சீனாவில் தோன்றிய திறந்த மூல AI நிறுவனம்; ஆழ்ந்த தேடல் / ஆய்வு என்ற கருத்தை அடிப்படையாகக் கொண்டது. |
| DeepThink | ஆழ்சிந்தனை | — | ஆழ் (deep) + சிந்தனை (thought). Chain-of-Thought போன்ற, AI மாதிரி படிப்படியாக ஆழ்ந்து சிந்திக்கும் ஏரண முறைகளுக்குப் பொதுச் சொல். |
| Dense Layer | அடர்த்தி அடுக்கு | முழு இணைப்பு அடுக்கு | ஒரு நரவலையில் முந்தைய அடுக்கின் அனைத்து நரம்பணுக்களுடனும் முழுமையாக இணைக்கப்பட்ட அடுக்கு. |
| Dense Model | அடர்த்தி மாதிரி | — | நரவலையில் உள்ள அனைத்து நரம்பணுக்களும் ஒன்றுக்கொன்று முழுமையாக இணைக்கப்பட்டு, அனைத்து எடைகளும் செயல்படும் நிலையில் உள்ள மாதிரி (சிதறல் மாதிரிக்கு எதிர்மாறானது). |
| Deterministic Output | திட்டவட்ட வெளியீடு | உறுதியான வெளியீடு | ஒரே உள்ளீட்டை எத்தனை முறை கொடுத்தாலும், எவ்வித மாறுபாடும் இன்றி ஒவ்வொரு முறையும் AI ஒரே மாதிரியான முடிவைத் தரும் தன்மை (Temperature 0 ஆக இருக்கும் நிலை). |
| Deterministic Reasoning | திட்டவட்ட ஏரணம் | — | மாறாத கணித விதிகளின்படி, ஒவ்வொரு முறையும் ஒரே துல்லியமான முடிவைத் தரும் ஏரண முறை. |
| Diffusion Model | விரவல் மாதிரி | பரவல் மாதிரி | விரவல் (diffusion) + மாதிரி (model). இரைச்சலைச் சேர்த்துப் பின் அதை நீக்குவதன் மூலம் தரவுகளிலிருந்து புதிய படங்களையோ ஒலிகளையோ உருவாக்கும் இயற்றறிவு மாதிரி. |
| Direct Preference Optimization (DPO) | நேரடி விருப்ப உகப்பாக்கம் | — | நேரடி (direct) + விருப்பம் (preference) + உகப்பாக்கம் (optimization). தனி ஒரு வெகுமதி மாதிரி இல்லாமல், மனித விருப்பத் தரவுகளிலிருந்து நேரடியாக மாதிரியைச் செப்பனிடும் எளிய முறை. |
| Distillation | அறிவுச் சுருக்கம் | சாரம் பிரித்தெடுத்தல் | பெரிய "ஆசிரிய" மாதிரியிலிருந்து அதன் அறிவைச் சிறிய "மாணவ" மாதிரிக்குப் பரிமாற்றும் மாதிரிச் சுருக்க நுட்பம். |
| Distribution Shift | தரவு நகர்ச்சி | விநியோக மாற்றம் | பயிற்சித் தரவும் பயன்பாட்டுத் தரவும் வேறுபடும் நிலை; மாதிரியின் செயல்திறன் நடைமுறைச் சூழலில் வீழ்ச்சியடையக் காரணம். |
| Documentation | ஆவணமாக்கம் | விளக்கக் குறிப்புகள் | மென்பொருள் அல்லது மாதிரியின் வடிவமைப்பு, பயன்பாடு மற்றும் குறிமுறைகளை விளக்கும் விரிவான எழுத்துப்பூர்வப் பதிவு. |
| Dropout | துளைவிடல் | நரம்புவிடுபடல் | பயிற்சியில் சில நரம்புகளைத் தற்காலிகமாக முடக்கி மிகைப்பொருத்தத்தைக் குறைக்கும் சீரமைப்பு நுட்பம். |
| Edge AI | விளிம்புநிலைச் செய்யறிவு | கருவிநிலைச் செய்யறிவு | விளிம்பு (edge) + நிலை (level) + செய்யறிவு. முகில் சேவையகங்களில் அல்லாமல், பயனரின் சாதனத்திலேயே (கைபேசி, IoT கருவி) நேரடியாக இயங்கும் AI மாதிரி. |
| Edit | தொகு | திருத்து | ஏற்கனவே உள்ள உரை, நிரல் அல்லது தரவை மாற்றி அமைக்கும் செயல். [^1] |
| Embeddings | உட்பொதிவுகள் | உள்ளடக்குகள் | உள் (inner) + பொதி (contain). சொற்கள் அல்லது தரவுகளை AI மாதிரி புரிந்துகொள்ளும் வகையில் எண்களாக (Vectors) மாற்றியமைக்கும் முறை. |
| Emergence | தோன்றுமியல்பு | — | தோன்றும் (arising) + இயல்பு (nature). சிறிய கூறுகளின் சேர்க்கையால், அவை தனித்தனியாகக் கொண்டிராத புதிய பண்புகள் தற்செயலாக வெளிப்படும் நிகழ்வு; வெளிப்படும் திறன்களின் அடிப்படைக் கருத்து. |
| Emergent Abilities | வெளிப்படும் திறன்கள் | தோன்றும் திறன்கள் | AI மாதிரியின் அளவு அதிகரிக்கும்போது, சிறிய மாதிரிகளில் இல்லாத புதிய திறன்கள் (உ-ம்: Instruction-following) தற்செயலாக வெளிப்படும் நிகழ்வு. |
| Emotion Classification | உணர்வு வகைப்பாடு | — | உணர்வு (emotion) + வகைப்பாடு (classification). ஒரு உள்ளீட்டிலிருந்து (உரை, படம், ஒலி) கோபம், மகிழ்ச்சி, துக்கம் போன்ற உணர்வுகளை அடையாளம் கண்டு பிரிக்கும் AI நுட்பம். |
| Emotion Detection | உணர்வுக் கண்டறிதல் | — | உணர்வு (emotion) + கண்டறிதல் (detection). மனிதனின் உணர்வு நிலையை உரை, முகபாவம், ஒலியின் சுருதி அல்லது உடற்சைகைகள் வழியாகக் கண்டறியும் AI செயல்பாடு. |
| Emotional Prompting | உணர்வுத் தூண்டுதல் | உணர்வுநிலைக் கட்டளை | AI மாதிரியின் வெளியீட்டில் குறிப்பிட்ட உணர்வை எழுப்பும் விதத்தில் தூண்டுவினா வடிவமைப்பு; AI-இன் உணர்வுத் தெரிவுத் திறனை மேம்படுத்த உதவுகிறது. |
| Encoder | குறியாக்கி | — | குறி (code) + ஆக்கி (maker). மனித உள்ளீட்டைக் கணினி புரிந்துகொள்ளும் உள்ளார்ந்த எண் தரவாக (Vector / Embedding) மாற்றும் பகுதி. |
| Environment Variable | சுற்றுப்புற மாறி | சூழல் மாறி | சுற்றுப்புறம் (environment) + மாறி (variable). ஒரு மென்பொருள் அல்லது கணினி இயங்கும் சூழல் சார்ந்த அமைப்பு மதிப்புகளைச் சேமித்து வைக்கும் மாறும் தரவு. [^1] |
| Epoch | சுழற்சி | முழுச் சுழற்சி | ஒரு AI மாதிரியை முழுப் பயிற்சித் தரவைப் பயன்படுத்தி ஒருமுறை பயிற்றுவித்தல்; பெரும்பாலான பயிற்சிகளில் பல சுழற்சிகள் தேவைப்படும். |
| Ethics in AI | செய்யறிவு நெறிமுறை | — | AI அமைப்புகளின் வடிவமைப்பு, பயன்பாடு மற்றும் தாக்கம் தொடர்பான நெறிமுறை, நேர்மை, பொறுப்புணர்வு மற்றும் சமூகச் சிக்கல்களை ஆராயும் துறை. |
| Evaluation | மதிப்பீடு | — | AI மாதிரியின் செயல்திறன், துல்லியம் மற்றும் பாதுகாப்பை அளவிடப் பயன்படுத்தப்படும் முறைகள் மற்றும் கருவிகள் (eval). |
| Explainability | விளக்கத் தகவு | — | விளக்கம் (explanation) + தகவு (capability). AI மாதிரியின் முடிவுகளை மனிதர்கள் புரிந்துகொள்ளக்கூடிய அளவில் விளக்கும் பண்பு. |
| Explainable AI (XAI) | விளக்கமளிக்கத்தக்க செய்யறிவு | — | AI மாதிரிகள் தங்கள் முடிவுகளை எவ்வாறு எடுத்தன என்பதை மனிதர்களுக்குப் புரியும் வகையில் விளக்கக்கூடிய திறன்; கருப்புப் பெட்டி (Black Box) சிக்கலுக்கான தீர்வு. |
| F1 Score | F1 மதிப்பெண் | — | துல்லிய விகிதம் (Precision), மீட்டெடுப்பு விகிதத்தின் (Recall) சமநிலைச் சராசரி. |
| Facial Action Coding System (FACS) | முகச்செயல் குறியீட்டு முறைமை | — | முகம் (face) + செயல் (action) + குறியீடு (coding) + முறைமை (system). ஏக்மன்-ஃபிரீசன் (1978) உருவாக்கிய முறைமை; முகத் தசை இயக்கங்களை எண்களாகக் குறியீடாக்கி, உணர்வுகளை அளவீட்டு வடிவத்தில் கண்டறியும் கருவி. |
| Facial Expression Recognition | முகபாவம் அறிதல் | முக உணர்வு அறிதல் | முகம் (face) + பாவம் (expression) + அறிதல் (recognition). முகப் படம் அல்லது காணொளியிலிருந்து உணர்வை வகைப்படுத்த உதவும் கணினியியல் பார்வை (Computer Vision) நுட்பம். |
| Fairness | நேர்மை | நீதிமை | AI முடிவுகள் குறிப்பிட்ட குழுக்களுக்குச் சாதகமாகவோ பாதகமாகவோ இல்லாமல், ஒருதலைச் சார்பின்றி நியாயமாக இருப்பதை உறுதிசெய்தல். |
| Feature (Mech Interp) | பண்பு | — | இயக்கமுறை விளக்கத்திறன் ஆய்வில், ஒரு நரவலையில் ஒரு குறிப்பிட்ட கருத்தை அல்லது வடிவத்தைக் குறிக்கும் தனிப்பட்ட திசையுருவாக்கம். |
| Feature Extraction | பண்பெடுப்பு | — | பண்பு (feature) + எடுப்பு (extraction). மூலத் தரவிலிருந்து பயனுள்ள பண்புகளை மட்டும் பிரித்தெடுக்கும் செயல்; இயந்திரக் கற்றலின் ஒரு முதன்மைப் படி. |
| Federated Learning | கூட்டமைப்புக் கற்றல் | ஒன்றியக் கற்றல் | தரவை மையத்திற்கு அனுப்பாமல், கருவிகளிலேயே பயிற்றுவித்துப் பயிற்சி முடிவுகளை மட்டும் ஒன்றிணைக்கும் பாதுகாப்பான கற்றல் முறை. |
| Feedforward Neural Networks | முன்னூட்ட நரவலை | — | முன் (forward) + ஊட்டம் (feed) + நரவலை. தகவல்கள் உள்ளீட்டு அடுக்கிலிருந்து வெளியீட்டு அடுக்குக்கு ஒரே திசையில் (முன்னோக்கி) மட்டுமே சுழற்சியின்றிப் பயணிக்கும் நரவலை. [^1] |
| Few-shot Prompting | சிறுமாதிரித் தூண்டுதல் | சில எடுத்துக்காட்டுத் தூண்டுதல் | AI-க்குக் கட்டளை இடும்போது, அது எவ்வாறு செயல்பட வேண்டும் என்பதற்காகச் சில எடுத்துக்காட்டுகளை உள்ளீட்டிலேயே வழங்கும் முறை. |
| Fine-Grained | நுண்தர | — | மிகச் சிறிய விவர நிலை; வகைப்பாட்டில் பல உட்பிரிவுகளுடன் விரிவாகப் பகுப்பாய்வு செய்தல். |
| Fine-tuning | நுண்திருத்தம் | சிறப்புப் பயிற்சி / செப்பனிடல் | நுண் (fine/subtle) + திருத்தம் (tuning). ஏற்கனவே பொதுவான தரவுகளில் கற்ற ஒரு பெரிய AI மாதிரிக்கு, குறிப்பிட்ட ஒரு துறை சார்ந்த தரவுகளை மட்டும் கொடுத்து மேலும் துல்லியமாக மாற்றுவதற்கான பயிற்சி. |
| Finite State Machine (FSM) | வரையறு நிலை இயந்திரம் | நிலை இயந்திரம் | குறிப்பிட்ட தூண்டுதல்களுக்கு ஏற்ப ஒரு நிலையிலிருந்து மற்றொரு நிலைக்கு மாறும் கணித மாதிரி; உரையாடல் சுழற்சிகளை மாதிரியாக்கம் செய்யப் பயன்படுகிறது. |
| Foundation Model | அடிப்படை மாதிரி | அடித்தள மாதிரி | பல்வேறு வேலைகளுக்கு அடித்தளமாக அமையும் பெரும் முன்பயிற்சி பெற்ற AI மாதிரி (எ.கா: GPT, Claude, Gemini). |
| Function Calling | சார்பு அழைப்பு | கருவி அழைப்பு | சார்பு (function) + அழைப்பு (call). LLM வெளிப்புறக் கருவிகளை (API, தரவுத்தளம்) நேரடியாக அழைக்கும் திறன். |
| Generative Adversarial Network (GAN) | இயற்றும் எதிரிடை வலை | எதிர்வலைப் பின்னல் | இயற்றும் (generative) + எதிரிடை (adversarial) + வலை (network). இரண்டு போட்டியிடும் நரவலைகள் (இயற்றி, கண்டறிவி) மூலம் புதிய தரவுகளை உருவாக்கும் கட்டமைப்பு. |
| Generative AI | இயற்றறிவு | ஈனும் செய்யறிவு | இயற்று (create/compose) + அறிவு (intelligence). புதிய உரை, படம், ஒலி அல்லது நிரல் குறிமுறைகளைத் தாமாகவே உருவாக்கும் திறன் கொண்ட AI மாதிரி. [^1] |
| Gradient | சரிவு | படிமச் சாய்வு | இழப்புச் சார்பின் (Loss Function) மாற்ற வேகம்; எந்தத் திசையில் அளபுருக்களை நகர்த்தினால் பிழை குறையும் என்பதைச் சுட்டிக் காட்டுகிறது. (கணித நூல்களில் "சரிவு" என்பதே நிறுவப்பட்ட சொல்.) |
| Gradient Descent | சரிவிறக்கம் | படிம இறக்கம் | சரிவு (gradient) + இறக்கம் (descent). ஒரு AI மாதிரியின் இழப்பைக் குறைக்க அதன் அளபுருக்களைப் படிப்படியாகச் சரிசெய்யும் உகப்பாக்க நெறிமுறை; ஆழ்கற்றலின் அடித்தளம். |
| Gram (in N-gram) | கிறுவம் | — | உரைப் பகுப்பாய்வில் (N-gram) தொடர்ச்சியாக வரும் உறுப்புகளின் (எழுத்து அல்லது சொல்) அடிப்படை அலகு. [^1] |
| GraphRAG | வரைபட RAG | வரைபட மீட்டெடுப்பு | அறிவு வரைபடத்தையும் RAG-ஐயும் ஒருங்கிணைத்து, தொடர்புகளின் அடிப்படையில் ஆழமான விடைகளை வழங்கும் கட்டமைப்பு. |
| Ground Truth | அடிப்படை உண்மை | மெய்த்தரவு | மாதிரியின் கணிப்பை ஒப்பிட்டு மதிப்பிடப் பயன்படும் சரியான விடை / அடையாளம். |
| Grounding | மெய்ப்பிணைப்பு | ஆதாரமூட்டல் | மெய் (truth) + பிணைப்பு (anchoring). AI பதிலை வெளிப்புற ஆதாரங்களுடன் (ஆவணங்கள், தரவுத்தளங்கள்) இணைக்கும் செயல்; மதிமயக்கத்தைக் குறைக்க உதவுகிறது. |
| Guardrails | பாதுகாப்பு வேலிகள் | நெறிமுறை வேலிகள் | AI மாதிரி தவறான, ஆபத்தான அல்லது இழிவான பதில்களைத் தராமல் தடுக்க அதன் உள்ளீட்டிலும் வெளியீட்டிலும் வைக்கப்படும் கட்டுப்பாட்டு அடுக்குகள். |
| Hallucination | மதிமயக்கம் | பொய் உருவாக்கம் | மதி (intellect) + மயக்கம் (delusion). AI தவறான தகவலை உண்மையைப் போல நம்பிக்கையாக உருவாக்கும் நிலை. [^1] |
| Hallucination Mitigation | மதிமயக்கத் தடுப்பு | — | மதிமயக்கம் (hallucination) + தடுப்பு (mitigation). AI மாதிரிகள் தவறான தகவல்களை உண்மையைப் போலச் சொல்வதைத் தடுக்க அல்லது குறைக்கப் பயன்படுத்தப்படும் நுட்பங்கள் (உ-ம்: RAG). |
| Hallucination Rate | மதிமயக்க வீதம் | — | ஒரு AI மாதிரி கொடுக்கும் மொத்தப் பதில்களில் எத்தனை விழுக்காடு மதிமயக்கப் பதில்கள் என அளவிடும் பிழை அளவீடு; அளவுகோலிலும் மதிப்பீட்டிலும் முக்கிய அளவீடு. |
| Hyperparameter | மேலளபுரு | கட்டுப்பாட்டு அளபுரு | மேல் (over/hyper) + அளபுரு (parameter). மாதிரி தானாகக் கற்றுக் கொள்ளாமல், பயிற்சிக்கு முன்பே வடிவமைப்பாளரால் முன்கூட்டியே அமைக்கப்படும் அளபுரு (உ-ம்: Learning Rate, Batch Size). |
| Image Generation | படவியற்றல் | பட உருவாக்கம் | படம் (image) + இயற்றல் (generation). உரைக் கட்டளைகள் அல்லது பிற உள்ளீடுகளின் அடிப்படையில் செய்யறிவைப் பயன்படுத்திப் புதிய படங்களை உருவாக்கும் செயல்முறை. |
| In-context Learning | சூழலுள் கற்றல் | — | சூழல் (context) + உள் (within) + கற்றல் (learning). எடுத்துக்காட்டுகளை தூண்டுவினாவிலேயே கொடுத்து, நுண்திருத்தமின்றி மாதிரி கற்கும் முறை. |
| Inductive Reasoning | தொகுப்பு ஏரணம் | — | தொகுப்பு (gathering/induction) + ஏரணம் (logic). குறிப்பிட்ட பல அவதானிப்புகள் அல்லது எடுத்துக்காட்டுகளை வைத்து, ஒரு பொதுவான விதியை அல்லது கணிப்பை உருவாக்கும் முறை (இயந்திரக் கற்றல் இதன் அடிப்படையில்தான் இயங்குகிறது). |
| Inference | முடிவுருவாக்கம் | அனுமானம் | முடிவு (conclusion) + உருவாக்கம் (creation). AI மாதிரியைப் பயிற்றுவித்த பிறகு, அதைப் பயன்படுத்திப் புதிய உள்ளீட்டிற்குக் கணிப்பு அல்லது பதில் வழங்கும் செயல்முறை (பயிற்சிக்கு எதிர்மாறானது). |

| Instruction Tuning | கட்டளை நுண்திருத்தம் | — | கட்டளை (instruction) + நுண்திருத்தம் (fine-tuning). பயனர் கட்டளைகளை AI மாதிரி துல்லியமாகப் பின்பற்ற உதவும் சிறப்பு நுண்திருத்தப் பயிற்சி. |
| Internet of Things (IoT) | பொருள்களின் பிணையம் | இணை பொருள்கள் | இணையத்துடன் இணைக்கப்பட்ட அன்றாடப் பொருள்கள் (உணர்வேற்பிகள், கருவிகள்) ஒன்றோடொன்று தரவைப் பகிரும் வலையமைப்பு. |
| Interpretability | விளக்கத்திறன் | — | விளக்கம் (interpretation) + திறன் (capability). மாதிரி எப்படி முடிவெடுக்கிறது என்பதைப் புரிந்துகொள்ளும் அளவு; விளக்கமளிக்கத்தக்க செய்யறிவுடன் தொடர்புடையது. |
| Iterative Refinement | மீள்சீரமைப்பு | சீரமைப்புச் சுழற்சி | மீள் (iterative/again) + சீரமைப்பு (refinement). ஒரு விடையையோ வெளியீட்டையோ பல சுழற்சிகளாகத் திரும்பத் திரும்பச் செம்மைப்படுத்தி மேம்படுத்தும் முறை. |
| Jailbreak | விதிவிலக்கு ஊடுருவல் | பாதுகாப்பு மீறல் தூண்டல் | விதிவிலக்கு (exception) + ஊடுருவல் (intrusion). AI மாதிரியின் உள்ளமைக்கப்பட்ட பாதுகாப்புக் கட்டுப்பாடுகளையும் அடிப்படை விதிகளையும் தந்திரமாக மீறச் செய்து, தடைசெய்யப்பட்ட பதில்களைப் பெறும் தாக்குதல் முறை. |
| K-skip-n-gram | த-தாவிய-எ-கிறுவம் | — | உரைப் பகுப்பாய்வில் 'த' (K) எண்ணிக்கையிலான சொற்களைத் தாண்டி அமையும் 'எ' (N) அளவிலான சொற்றொடர் அமைப்பு. [^1] |
| Knowledge Base | அறிவுத்தளம் | — | அறிவு (knowledge) + தளம் (base). ஒரு குறிப்பிட்ட துறை சார்ந்த தகவல்கள், தரவுகள் மற்றும் விதிகளைச் சேமித்து வைத்து விடையளிக்க உதவும் மையக் கட்டமைப்பு. |
| Knowledge Cutoff | அறிவு வரம்பு | அறிவெல்லை | அறிவு (knowledge) + வரம்பு (cutoff). AI மாதிரி பயிற்றுவிக்கப்பட்ட தரவின் கடைசித் தேதி; அதற்குப் பிறகான நிகழ்வுகள் மாதிரிக்குத் தெரியாது. |
| Knowledge Graph | அறிவு வரைபடம் | அறிவுத் தொடர்பமைப்பு | உண்மை உலகப் பொருள்கள், நிகழ்வுகள் அல்லது கருத்துகளுக்கு இடையிலான உறவுகளை வலைப்பின்னல் போன்று இணைத்துச் சேமிக்கும் கட்டமைப்பு. |
| KV Cache | விசை-மதிப்புத் தேக்ககம் | வி-ம தேக்ககம் | விசை (key) + மதிப்பு (value) + தேக்ககம் (cache). மாற்றுநர் (Transformer) மாதிரிகளில், முந்தைய சொல்துண்டுகளின் விசை மற்றும் மதிப்பு தரவுகளை நினைவகத்தில் சேமித்து வைத்து, உரை உருவாக்கத்தை வேகப்படுத்தும் நுட்பம். |
| Language Action Model (LAM) | மொழிச்செயல் மாதிரி | — | மொழி (language) + செயல் (action) + மாதிரி (model). மொழியைப் புரிந்துகொள்வது மட்டுமின்றி, மென்பொருள்களில் நேரடியாகச் செயல்களை நிறைவேற்றும் திறன் கொண்ட AI மாதிரி. [^1] |
| Language Model (LM) | மொழிமாதிரி | மொழியொப்பாக்கம் | சொற்களின் தொடர்ச்சியைக் கணித்து, மனித மொழியைப் புரிந்துகொள்ளவும் உருவாக்கவும் வடிவமைக்கப்பட்ட AI கட்டமைப்பு. [^1] |
| Large Language Model (LLM) | பெருமொழி மாதிரி | மாமொழி மாதிரி | பெரு (large) + மொழி (language) + மாதிரி (model). பில்லியன் கணக்கான அளபுருக்களையும் பரந்த தரவுகளையும் கொண்டு பயிற்றுவிக்கப்பட்ட மிகப் பெரிய மொழி மாதிரி. [^1] |
| Latency | செயல்தாமதம் | பின்னடைவு | செயல் (action) + தாமதம் (delay). பயனர் கோரிக்கைக்கும் AI மாதிரி அளிக்கும் பதிலுக்கும் இடையிலான நேர இடைவெளி; உண்மை நேர AI பயன்பாடுகளில் இது மிகக் குறைவாக இருக்க வேண்டும். |
| Latent Concept Model (LCM) | உள்ளுறைக் கருத்துரு மாதிரி | — | உள்ளுறை (latent/hidden) + கருத்துரு (concept) + மாதிரி (model). தரவுகளில் வெளிப்படையாகத் தெரியாத, ஆனால் உள்ளீடுகளுக்குப் பின்னால் மறைந்திருக்கும் அடிப்படைக் கருத்துகளைக் கண்டறியும் மாதிரி. [^1] |
| Latent Space | உள்ளுறை வெளி | மறை வெளி | உள்ளுறை (latent) + வெளி (space). மாதிரி தரவின் அடிப்படைப் பண்புகளைச் சுருக்கிச் சேமிக்கும் எண் வெளி. |
| Latent Variable | உள்ளுறை மாறி | மறை மாறி | உள்ளுறை (latent/hidden) + மாறி (variable). தரவில் நேரடியாகத் தெரியாமல், மற்ற மாறிகள் வழியாக மட்டுமே ஊகிக்கக்கூடிய மறைந்த மாறி. |
| Layer | அடுக்கு | — | நரவலையின் ஒரு செயல்பாட்டு நிலை; உள்ளீட்டு அடுக்கு, மறை அடுக்குகள் மற்றும் வெளியீட்டு அடுக்கு எனப் பல நிலைகளைக் கொண்டது. |
| Leaderboard | முன்னிலைப் பட்டியல் | தரவரிசைப் பட்டியல் | முன்னிலை (leading) + பட்டியல் (list). வெவ்வேறு AI மாதிரிகளின் செயல்திறன் அளவீடுகளை மதிப்பெண் அடிப்படையில் வரிசைப்படுத்திக் காட்டும் தரவரிசைப் பட்டியல். |
| Learning Objective | கற்றல் நோக்கம் | — | இயந்திரக் கற்றலின் போது ஒரு மாதிரி அடைய வேண்டிய இலக்கு அல்லது குறைக்க வேண்டிய பிழை அளவு. |
| Learning Rate | கற்றல் வீதம் | — | கற்றல் (learning) + வீதம் (rate). எடைகள் எவ்வளவு வேகமாகப் புதுப்பிக்கப்படுகின்றன என்பதைக் கட்டுப்படுத்தும் மேலளபுரு. |
| Logical Reasoning | ஏரணப் பகுத்தறிதல் | ஏரண முடிவுருவாக்கம் | ஏரணம் (logic) + பகுத்தறிதல் (reasoning). தரவுகளின் அடிப்படையில் காரண காரியங்களை ஆராய்ந்து ஏரண முறையில் முடிவெடுக்கும் திறன். |
| Logical Schema | ஏரண வரைவு | ஏரண மாதிரி | ஏரணம் (logic) + வரைவு (schema). தரவுகள் மற்றும் விதிகளுக்கு இடையே உள்ள தொடர்புகளை முறையான ஏரண அமைப்பாக வடிவமைக்கும் வரைபடம். |
| Long Context | நீண்ட சூழல் | நீண்ட உள்ளீட்டு வரம்பு | மிக நீளமான உள்ளீட்டை (லட்சக்கணக்கான சொல்துண்டுகள்) ஒரே நேரத்தில் கையாளும் மாதிரித் திறன். |
| Loss Function | இழப்புச் சார்பு | பிழை அளவீடு | இழப்பு (loss) + சார்பு (function). AI மாதிரியின் கணிப்புகளுக்கும் உண்மை மதிப்புகளுக்கும் இடையே உள்ள வேறுபாட்டை அளவிடும் கணிதச் சார்பு; பயிற்சியின் போது இதைக் குறைப்பதே முக்கிய நோக்கமாகும். |
| Low-Rank Adaptation (LoRA) | குறைவரிசைத் தழுவல் | குறை-சரிசெய்தல் சேர்க்கை | குறை (low) + வரிசை (rank) + தழுவல் (adaptation). முழு மாதிரியையும் மாற்றாமல், குறைந்த அளவிலான கூடுதல் எடைகளை (Low-Rank Matrices) மட்டும் இணைத்துப் பயிற்றுவிக்கும் எடை-சிக்கன நுண்திருத்த (PEFT) முறை. |
| Machine Learning (ML) | இயந்திரக் கற்றல் | — | இயந்திரம் (machine) + கற்றல் (learning). வெளிப்படையான நிரலாக்கம் இல்லாமல், தரவிலிருந்து தானாகவே கற்று வடிவங்களைக் கண்டறிந்து கணிப்புகளை வழங்கும் கணினியியல் துறை. |
| Machine Translation | இயந்திர மொழிபெயர்ப்பு | — | இயந்திரம் (machine) + மொழிபெயர்ப்பு (translation). ஒரு மொழியிலிருந்து மற்றொரு மொழிக்குக் கணினி அல்லது AI மூலம் நேரடியாக மொழிபெயர்க்கும் நுட்பம். |

| MCP (Model Context Protocol) | மாதிரிச் சூழல் நெறிமுறை | மாதிரிப் பின்புல நெறிமுறை | மாதிரி (model) + சூழல் (context) + நெறிமுறை (protocol). AI மாதிரிகள் வெளிப்புறக் கருவிகள் / தரவுத்தளங்களுடன் பாதுகாப்பாகத் தொடர்புகொள்ளும் தரப்படுத்தப்பட்ட நெறிமுறை. |
| Mechanistic Interpretability | இயக்கமுறை விளக்கத்திறன் | — | இயக்கம் (mechanism) + முறை (method) + விளக்கம் (interpretation) + திறன் (capability). | Memory (AI context) | நினைவகம் | உரையாடல் நினைவகம் | AI உரையாடல் வரலாறு அல்லது நீண்டகாலத் தகவலைச் சேமித்துப் பயன்படுத்தும் திறன். |

| Model Weights | மாதிரி எடைகள் | அளபுரு எடைகள் | ஒரு AI மாதிரி பயிற்சியின் போது தரவுகளிலிருந்து தானாகவே கற்றுக் கொள்ளும் எண்ணியல் மதிப்புகள். |

| Multimodal AI | பன்முகச் செய்யறிவு | பன்முக மாதிரி | பன் (multi) + முகம் (mode/facet) + செய்யறிவு (AI). உரை, படம், ஒலி, காணொளி போன்ற பல வகை உள்ளீடுகளைக் கையாளும் திறன் கொண்ட AI மாதிரி. |
| N-gram | எ-கிறுவம் | — | உரைப் பகுப்பாய்வில் 'எ' (N) எண்ணிக்கையிலான தொடர்ச்சியான உறுப்புகளின் (எழுத்துகள் அல்லது சொற்கள்) அமைப்பு. [^1] |
| N-gram Language Modelling | எ-கிறுவ மொழியொப்பாக்கம் | எ-கிறுவ மொழி மாதிரி | முந்தைய (N-1) சொற்களைப் பயன்படுத்தி அடுத்த சொல்லைக் கணிக்கும் ஒரு பாரம்பரிய உரை மாதிரி நுட்பம். [^1] |
| Named Entity Recognition (NER) | பெயர்ச்சொல்லுணர்தல் | பெயருரு அறிதல் | பெயர்ச்சொல் (proper noun) + உணர்தல் (recognition). உரையில் உள்ள மக்கள், இடங்கள், நிறுவனங்கள் போன்ற குறிப்பிட்ட பெயர்களைத் தானாகவே அடையாளம் கண்டு வகைப்படுத்தும் முறை. [^1] |
| Natural Language Processing (NLP) | இயன்மொழியாள்கை | இயற்கை மொழிச் செயலாக்கம் | இயன் (natural) + மொழி (language) + ஆள்கை (processing). மனித மொழியைக் கணினிகள் புரிந்துகொள்ள, பகுப்பாய்வு செய்ய மற்றும் உருவாக்க உதவும் செய்யறிவுப் பிரிவு. [^1] |
| Neural Networks | நரவலை | நரம்பணு வலைப்பின்னல் | நரம்பு (nerve) + வலை (network). மனித மூளையின் கட்டமைப்பைப் பின்பற்றித் தரவுகளில் உள்ள சிக்கலான வடிவங்களைக் கற்றுக் கொள்ளும் கணினி அமைப்பு. [^1] |
| Neural Search | நரவலைத் தேடல் | பொருள்சார் தேடல் | நரவலை (neural network) + தேடல் (search). பாரம்பரியத் தேடல் போலக் குறிப்பிட்ட வார்த்தைகளை மட்டும் தேடாமல், நரவலைகளைப் பயன்படுத்திப் பயனர் கேட்கும் கேள்வியின் ஆழமான பொருளைப் புரிந்துகொண்டு தேடும் நுட்பம். |
| Ngram Vocabulary | கிறுவடைவு | எ-கிறுவ அகராதி | கிறுவம் (gram) + அடைவு (vocabulary). பயிற்சித் தரவிலிருந்து சேகரிக்கப்பட்ட அனைத்து எ-கிறுவங்களின் முழுமையான பட்டியல் அல்லது அகராதி. [^1] |
| OCR (Optical Character Recognition) | ஒளியியல் எழுத்துணர்தல் | ஒளிவழி எழுத்துணர்தல் | ஒளியியல் (optical) + எழுத்து (character) + உணர்தல் (recognition). அச்சிடப்பட்ட அல்லது கையால் எழுதப்பட்ட ஆவணங்களைப் படம்பிடித்து, அதிலுள்ள எழுத்துகளைக் கணினி திருத்தக்கூடிய உரையாக மாற்றும் நுட்பம். |
| On-device AI | கருவியகச் செய்யறிவு | சாதனவழிச் செய்யறிவு | கருவியகம் (within-device) + செய்யறிவு. இணையத் தொடர்பின்றி, கருவியின் (கைபேசி போன்றவை) சொந்த வன்பொருளைப் பயன்படுத்தியே தரவுகளைச் செயலாக்கும் AI அமைப்பு. |

| Parameter | அளபுரு | எடை | மாதிரி கற்றுக் கொள்ளும் எண்ணியல் மதிப்பு (Weights). ஒரு பெருமொழி மாதிரியில் பில்லியன் கணக்கான அளபுருக்கள் இருக்கலாம். |

| Prompt | தூண்டுவினா | கட்டளை | தூண்டு (prompt/trigger) + வினா (question). AI-யிடம் வேலை வாங்க நாம் கொடுக்கும் உள்ளீடு அல்லது அறிவுறுத்தல். |
| Prompt Chaining | தூண்டுவினாச் சங்கிலி | தூண்டுவினா இணைப்பு | தூண்டுவினா (prompt) + சங்கிலி (chain). ஒரு தூண்டுவினாவின் வெளியீட்டை அடுத்த தூண்டுவினாவுக்கு உள்ளீடாகத் தொடர்ந்து இணைத்து, சிக்கலான பல படிநிலைப் பணிகளை நிறைவேற்றும் முறை. |
| Prompt Engineering | தூண்டுவினாவியல் | தூண்டுவினா நுட்பம் | தூண்டுவினா (prompt) + இயல் (field of study). AI மாதிரியிடமிருந்து சிறந்த பதிலைப் பெறுவதற்காக உள்ளீட்டுக் கட்டளைகளைத் துல்லியமாக வடிவமைக்கும் முறை. [^1] |
| Prompt Injection | தூண்டுவினா ஊடுருவல் | கட்டளை ஊடுருவல் | தூண்டுவினா (prompt) + ஊடுருவல் (injection). AI-க்கு வழங்கப்படும் உள்ளீட்டில் தீங்கான கட்டளைகளைச் செருகி, அதன் அடிப்படை விதிகளை மீறச் செய்யும் தாக்குதல் முறை. |
| Pruning | வெட்டித்திருத்தம் | கிளை நீக்கம் | வெட்டு (prune) + திருத்தம் (refinement). மாதிரிச் சுருக்க நுட்பம்; தேவையற்ற அல்லது குறைந்த முக்கியத்துவம் கொண்ட எடைகளை நீக்கி, மாதிரியின் அளவையும் கணக்கீட்டுச் சுமையையும் குறைக்கும் முறை. |
| Quantization | எண்ணளவாக்கம் | துல்லியச் சுருக்கம் | எண் (number) + அளவு (size) + ஆக்கம் (process). மாதிரியின் எடைகளை உயர் துல்லியத்திலிருந்து (32-bit) குறைந்த துல்லியத்திற்கு (8/4-bit) சுருக்கி, நினைவகமும் வேகமும் மேம்படுத்தும் நுட்பம். |
| Question Answering (QA) | வினா-விடை | — | கொடுக்கப்பட்ட சூழலிலிருந்து கேள்விக்கு விடை கண்டறியும் இயன்மொழியாள்கைப் பணி. |
| Reasoning Chain | ஏரணச் சங்கிலி | பகுத்தறிவுச் சங்கிலி | ஏரணம் (logic) + சங்கிலி (chain). சிந்தனைச் சங்கிலி (CoT) முறைக்கு மாற்றுச் சொல்; ஒரு முடிவை எட்டுவதற்காக AI மாதிரி பின்பற்றும் ஏரண முறை படிநிலைகளின் தொடர். |
| Reasoning Model | ஏரண மாதிரி | பகுத்தறிவு மாதிரி | ஏரணம் (logic) + மாதிரி (model). தகவல்களை வெறுமனே மனப்பாடம் செய்யாமல், காரண காரியங்களை ஆராய்ந்து ஏரண முறையில் முடிவெடுக்கும் திறன் கொண்ட AI மாதிரி. |
| Reasoning Tokens | ஏரணச் சொல்துண்டுகள் | சிந்தனைச் சொல்துண்டுகள் | ஏரணம் (logic) + சொல்துண்டுகள் (tokens). ஏரண மாதிரிகள் (o1, o3, R1) இறுதி விடை அளிக்கும் முன், உள்நிலையில் சிந்திக்கப் பயன்படுத்தும் சொல்துண்டுகள்; இவை பயனருக்குப் பெரும்பாலும் தெரியாது. |
| Recall | மீட்டெடுப்பு விகிதம் | — | மீட்டெடுப்பு (recall) + விகிதம் (ratio). தரவில் உள்ள உண்மையானவற்றில் எத்தனை மாதிரியால் கண்டறியப்பட்டன என்பதைக் காட்டும் அளவீடு. |
| Recurrent Neural Network (RNN) | பின்னூட்ட நரவலை | மீள்சுற்று நரவலை | பின்னூட்டம் (recurrent/feedback) + நரவலை. முந்தைய வெளியீட்டைத் தற்போதைய உள்ளீட்டின் ஒரு பகுதியாகப் பயன்படுத்தித் தொடர்ச்சியான தரவுகளைக் (மொழி, நேரம்) கையாளும் நரவலை. [^1] |
| Red Teaming | சிவப்பு அணி சோதனை | — | AI மாதிரியின் பாதுகாப்பு, பிழைகள் மற்றும் சாய்வுகளைக் கண்டறியத் திட்டமிட்ட தாக்குதல் அல்லது சவால் கேள்விகள் மூலம் சோதனை செய்யும் முறை. |
| Refusal | மறுப்பு | — | AI மாதிரி ஒரு கோரிக்கையை — பாதுகாப்பு, நெறிமுறை, அல்லது திறன் காரணங்களால் — நிறைவேற்ற முடியாது என்று வெளிப்படையாக மறுக்கும் வெளியீடு. |
| Regularization | சீரமைப்பு | ஒழுங்குமுறைப்படுத்தல் | மிகைப்பொருத்தத்தைக் குறைக்கப் பயிற்சியில் கட்டுப்பாடுகள் விதிக்கும் நுட்பங்கள் (Dropout, Weight Decay). |
| Reinforcement Learning (RL) | வலுவூட்டல் கற்றல் | — | வலு (strength) + ஊட்டல் (feeding) + கற்றல் (learning). ஒரு செயல்முகவர் சூழலோடு தொடர்புகொண்டு, செயல்களுக்கான வெகுமதிகள் அல்லது தண்டனைகள் மூலம் தாமாகவே கற்றுக் கொள்ளும் இயந்திரக் கற்றல் முறை. |
| Reinforcement Learning from Human Feedback (RLHF) | மனிதப் பின்னூட்ட வலுவூட்டல் கற்றல் | மனித வெகுமதி வலுவூட்டல் கற்றல் | மனித விருப்பங்களுக்கு ஏற்ப AI-இன் வெளியீடுகளைச் சீரமைக்கும் சிறப்புப் பயிற்சி முறை. |
| Relation Extraction | தொடர்பு பிரித்தெடுத்தல் | உறவு கண்டறிதல் | தொடர்பு (relation) + பிரித்தெடுத்தல் (extraction). உரையில் கண்டறியப்பட்ட இரண்டு பொருள்களுக்கு அல்லது நபர்களுக்கு இடையே உள்ள உறவைத் தானாகவே கண்டறிந்து வகைப்படுத்தும் முறை. |
| Responsible AI | பொறுப்பான செய்யறிவு | — | நேர்மை, வெளிப்படைத்தன்மை, பொறுப்புணர்வுடன் வடிவமைக்கப்படும் AI. |
| Retrieval-Augmented Generation (RAG) | மீட்டெடுப்பு-மேம்படுத்திய இயற்றல் | மீட்டெடுப்புசார் உரை உருவாக்கம் | மீட்டெடுப்பு (retrieval) + மேம்படுத்திய (augmented) + இயற்றல் (generation). வெளிப்புறத் தரவுத்தளங்களிலிருந்து தகவல்களை மீட்டெடுத்துத் துல்லியமான பதிலை உருவாக்கும் கட்டமைப்பு. |
| Reward Model | வெகுமதி மாதிரி | பாராட்டு மாதிரி | வெகுமதி (reward) + மாதிரி (model). AI தரும் பதில்களில் எது சிறந்தது என்று மனிதனைப் போலவே மதிப்பிட்டுப் புள்ளிகள் வழங்கும் துணைக் கணினி மாதிரி (RLHF-இல் பயன்படுகிறது). |
| RL from AI Feedback (RLAIF) | AI பின்னூட்ட வலுவூட்டல் கற்றல் | — | RLHF-ன் மாற்று முறை; மனிதர்களின் பின்னூட்டத்திற்குப் பதிலாக மற்றொரு AI மாதிரியின் கருத்துகளைப் பயன்படுத்தி வலுவூட்டல் கற்றலை மேற்கொள்ளும் முறை (Anthropic Constitutional AI-ல் பயன்படுகிறது). |
| Robustness | வலுவுறுதி | உறுதித்தன்மை | வலு (strength) + உறுதி (firmness). எதிர்பாராத உள்ளீடுகள், இரைச்சல், அல்லது தாக்குதல்கள் (Adversarial Attacks) ஏற்படும்போதும் AI மாதிரி நிலையாக, துல்லியமாகச் செயல்படும் திறன். |
| Sampling Strategy | மாதிரி எடுத்தல் உத்தி | சொல் தேர்வு முறை | AI மாதிரி அடுத்த சொல்லைத் தேர்ந்தெடுக்கும்போது பயன்படுத்தும் ஏரண முறை வழிமுறை (உ-ம்: Top-K, Top-P, Temperature). |
| Scalar | ஒற்றை மதிப்பு | தன்மதிப்பு | ஒற்றை (single) + மதிப்பு (value). ஒரே ஒரு எண்ணைக் கொண்ட மதிப்பு; திசையன், பன்பரிமாண அணி ஆகியவற்றுக்கு எதிர்மறை. |
| Segment Anything Model (SAM) | கூறு ஏதேனும் மாதிரி | ஏதும்கூறு மாதிரி | கூறு (segment) + ஏதேனும் (anything) + மாதிரி (model). படத்திலுள்ள எந்தவொரு பொருளையும் முன்கூட்டியே அறியாமலேயே துல்லியமாகப் பிரித்து (Segment) அடையாளம் காணும் கணினியியல் பார்வை மாதிரி. [^1] |
| Self-Attention | தன்கவனம் | தன்கவனிப்பு | தன் (self) + கவனம் (attention). உள்ளீட்டின் ஒவ்வொரு பகுதியும், அதே உள்ளீட்டின் மற்ற பகுதிகளுடன் எவ்வாறு தொடர்புபடுகிறது என்பதைக் கணக்கிடும் வழிமுறை; மாற்றுநரின் (Transformer) அடிப்படை. |
| Self-Consistency | தன்னொத்திசைவு | தன்முறை ஒருமை | தன் (self) + ஒத்திசைவு (consistency). சிந்தனைச் சங்கிலியில் (CoT) ஒரே கேள்விக்குப் பல சிந்தனைப் பாதைகளை உருவாக்கி, அதிகம் ஒத்திசைந்த விடையை இறுதி முடிவாகத் தேர்ந்தெடுக்கும் நுட்பம். |
| Self-supervised Learning | தன்மேற்பார்வைக் கற்றல் | — | தன் (self) + மேற்பார்வை (supervision) + கற்றல் (learning). அடையாளமிடாத தரவிலிருந்து தானே மேற்பார்வைக் குறிப்புகளை உருவாக்கிக் கற்கும் முறை (BERT, GPT). |
| Semantic Kernel | பொருள்சார் அகடு | கருத்தியல் கருவம் | பொருள் (meaning) + சார் (related) + அகடு (core/kernel). பாரம்பரிய நிரலாக்க மொழிகளுடன் (C#, Python) பெருமொழி மாதிரிகளை எளிதாக இணைக்க மைக்ரோசாப்ட் உருவாக்கிய திறந்த மூலக் கட்டமைப்பு. |
| Semantic Textual Similarity | பொருள்சார் உரை ஒப்புமை | — | பொருள் (meaning) + சார் (related) + உரை (text) + ஒப்புமை (similarity). இரண்டு வெவ்வேறு வாக்கியங்கள் ஒரே வார்த்தைகளைக் கொண்டிராவிட்டாலும், அவற்றின் ஒட்டுமொத்தப் பொருள் எவ்வளவு ஒத்துப் போகிறது என்பதை எண்களால் அளவிடுதல். |
| Semi-supervised Learning | அரைமேற்பார்வைக் கற்றல் | — | அரை (half/semi) + மேற்பார்வை (supervision) + கற்றல் (learning). சிறிது அடையாளமிட்ட தரவும் பெரும்பான்மை அடையாளமிடாத தரவும் கலந்த கற்றல் முறை. |
| Sentiment Analysis | உணர்வுப் பகுப்பாய்வு | — | உணர்வு (sentiment) + பகுப்பாய்வு (analysis). ஒரு உரையின் மொத்த உணர்வுச் சார்பை (நேர்மறை, எதிர்மறை, நடுநிலை) அளவிடும் இயன்மொழியாள்கை (NLP) நுட்பம். |
| Sentiment Detection | உணர்நிலை கண்டறிதல் | மனப்பாங்கைக் கண்டறிதல் / சொற்சுவையறிதல் | உணர் (feel) + நிலை (state) + கண்டறிதல் (detection). வார்த்தைகளில் பொதிந்துள்ள உள்நோக்கத்தையோ உணர்நிலையையோ துல்லியமாகக் கண்டறியும் செயல்முறை. |

| Server | வழங்கி | சேவையகம் | தரவுகளைச் சேமித்து வைத்து, வலையமைப்பில் உள்ள பிற கணினிகளுக்கு (Clients) சேவைகளை அல்லது தகவல்களை வழங்கும் மையக் கணினி. [^1] |
| Skipgram | தாவிய-கிறுவம் | — | தாவிய (skipped) + கிறுவம் (gram). உரைப் பகுப்பாய்வில் ஒரு குறிப்பிட்ட மையச் சொல்லை வைத்து, அதைச் சுற்றியுள்ள (முன் மற்றும் பின்) சொற்களைக் கணிக்கும் மாதிரி. [^1] |

| Softmax | மென்மை-அதிகபடி சார்பு | — | நரவலையின் வெளியீட்டு மதிப்புகளை நிகழ்தகவு விநியோகமாக மாற்றும் கணிதச் செயல்பாடு; வகைப்பாட்டில் பயன்படுகிறது. |
| Sparse Autoencoder (SAE) | சிதறல் தான்குறியாக்கி | — | சிதறல் (sparse) + தான் (self) + குறியாக்கி (encoder). இயக்கமுறை விளக்கத்திறன் ஆய்வில், ஒரு நரவலையின் தூண்டல்களைப் புரிந்துகொள்ளக்கூடிய தனிப்பட்ட பண்புகளாகச் சிதைத்துப் பிரிக்கும் கட்டமைப்பு (Anthropic-ன் முக்கிய ஆராய்ச்சிக் கருவி). |
| Sparse Model | சிதறல் மாதிரி | அரிது மாதிரி | சிதறல் (sparse) + மாதிரி (model). பல எடைகள் (Weights) பூஜ்யத்திற்கு அருகில் சிதறிக் காணப்படும் மாதிரி; இதனால் குறைந்த கணக்கீட்டுத் திறனிலேயே செயல்பட முடிகிறது. |
| Speculative Decoding | ஊகக் குறிநீக்கம் | முன்மதிப்பீட்டு விடையாக்கம் | ஊகம் (speculation) + குறிநீக்கம் (decoding). சிறிய மாதிரி ஊகிக்க, பெரிய மாதிரி சரிபார்க்க — இரட்டை மாதிரி வேகமூட்டும் நுட்பம். |
| Speech Recognition | பேச்சுணர்தல் | குரலறிதல் | பேச்சு (speech) + உணர்தல் (recognition). மனிதப் பேச்சைக் கணினி அல்லது மென்பொருள் உணர்ந்து புரிந்துகொள்ளும் தொழில்நுட்பம். |
| Speech-to-Text (STT) | பேச்சு-முதல்-உரை | ஒலி-உரை மாற்றம் | மனிதப் பேச்சை அல்லது ஒலியைப் புரிந்துகொண்டு அதை எழுத்துப்பூர்வமான உரையாக மாற்றும் நுட்பம். |
| Stable Diffusion | நிலைத்த விரவல் | சீரான விரவல் | நிலைத்த (stable) + விரவல் (diffusion). குறைந்த கணக்கீட்டுத் திறனில் இயங்கக்கூடிய, உரையிலிருந்து படங்களை உருவாக்கும் ஒரு திறந்த மூல விரவல் மாதிரி. |
| Steering | வழிநடத்தல் | திசை திருப்பல் | இயக்கமுறை விளக்கத்திறன் நுட்பங்களைப் பயன்படுத்தி, AI மாதிரியின் வெளியீட்டை ஒரு குறிப்பிட்ட பண்பு (உ-ம்: பாதுகாப்பு, நேர்மை) நோக்கிக் கட்டுப்படுத்தி வழிநடத்தும் முறை. |

| Summarization | சுருக்கமிடுதல் | உரைச் சுருக்கம் | நீண்ட உரையின் முக்கிய கருத்துகளைச் சிறிய வடிவில் தரும் இயன்மொழியாள்கைப் பணி. |

| Sycophancy | முகதுதி | துதிச் சார்பு | முகம் (face) + துதி (flattery). AI மாதிரி பயனரின் கருத்துகளுடன் அளவுக்கு மீறி ஒத்துப்போய், உண்மையான தகவலையோ எதிர்க் கருத்தையோ வழங்காமல் இருக்கும் தவறான நடத்தை. |

| Tool Calling | கருவி அழைப்பு | — | கருவி (tool) + அழைப்பு (call). AI மாதிரி வெளிப்புறக் கருவிகளை (API, தேடுபொறி, தரவுத்தளம்) நேரடியாக அழைத்து, அவற்றின் முடிவுகளை வெளியீட்டில் இணைக்கும் திறன். |
| Tool Use | கருவிப் பயன்பாடு | — | கருவி (tool) + பயன்பாடு (use). AI மாதிரி தனது மொழித் திறனைத் தாண்டி, வெளிப்புறக் கருவிகளை (கால்குலேட்டர், தேடுபொறி, குறிமுறை இயக்கி) பயன்படுத்திச் செயல் ஆற்றும் பரந்த திறன். |
| Toolformer | கருவியமைப்பி | கருவி-மாற்றுநர் | கருவி (tool) + அமைப்பி (former). கால்குலேட்டர்கள், தேடுபொறிகள் போன்ற வெளிப்புறக் கருவிகளை எப்பொழுது, எப்படிப் பயன்படுத்த வேண்டும் என்று தானாகவே கற்றுக் கொள்ளும் மொழி மாதிரி. |
| Top-K Sampling | உயர்-K மாதிரி எடுத்தல் | — | அடுத்த சொல்லைத் தேர்ந்தெடுக்கும்போது அதிக நிகழ்தகவு கொண்ட முதல் 'K' எண்ணிக்கையிலான சொற்களை மட்டும் கட்டுப்படுத்தும் முறை. |
| Top-P (Nucleus Sampling) | அணுகுத் தொகுதி மாதிரி எடுத்தல் | தொகுப்பு வரம்பு | அடுத்த சொல்லைத் தேர்ந்தெடுக்கும்போது குறிப்பிட்ட விழுக்காடு ஒட்டுமொத்த நிகழ்தகவு கொண்ட சொற்களை மட்டும் கணக்கில் கொள்ளும் முறை. |
| Transfer Learning | இடமாற்றக் கற்றல் | பரவல் கற்றல் | இடமாற்றம் (transfer) + கற்றல் (learning). ஒரு பணிக்காகப் பயிற்றுவிக்கப்பட்ட மாதிரியின் அறிவை, மற்றொரு தொடர்புடைய பணிக்குப் பயன்படுத்தும் முறை; அடிப்படை மாதிரிகளின் அடித்தளம். |
| Transformer | மாற்றுநர் | மாற்றி | தன்கவன (Self-Attention) வழிமுறையைப் பயன்படுத்தித் தொடர்ச்சியான தரவுகளை (குறிப்பாக மொழியை) இணையாகச் செயலாக்கும் ஆழ்கற்றல் நரவலைக் கட்டமைப்பு (உ-ம்: GPT, BERT). |
| Tree of Thoughts (ToT) | சிந்தனை மரம் | — | சிந்தனைச் சங்கிலியின் (CoT) விரிவாக்கம்; ஒரு சிக்கலைத் தீர்க்கப் பல சிந்தனைப் பாதைகளை மரக் கட்டமைப்பில் ஒருங்கே ஆராய்ந்து, சிறந்ததைத் தேர்ந்தெடுக்கும் முறை. |
| Unsupervised Learning | மேற்பார்வையற்ற கற்றல் | — | மேற்பார்வை (supervision) + அற்ற (without) + கற்றல் (learning). அடையாளமிடப்படாத தரவுகளிலிருந்தே வடிவங்கள், கொத்துகள் மற்றும் அமைப்புகளைத் தானாகவே கண்டறியும் இயந்திரக் கற்றல் முறை. |
| User Prompt | பயனர் தூண்டுவினா | பயனர் கட்டளை | பயனர் (user) + தூண்டுவினா (prompt). பயனர் AI-க்கு வழங்கும் கேள்வி அல்லது கட்டளை (அமைப்புத் தூண்டுவினாவுடன் இணைந்து செயல்படுகிறது). |
| User Role | பயனர் பாத்திரம் | — | பயனர் (user) + பாத்திரம் (role). உரையாடல் கட்டமைப்பில் பயனரின் உள்ளீட்டுப் பகுதியைக் குறிக்கும் பாத்திரம். |
| Validation Set | சரிபார்ப்புத் தொகுதி | — | சரிபார்ப்பு (validation) + தொகுதி (set). பயிற்சியின் போது மாதிரியின் செயல்திறனை மதிப்பிடப் பயன்படுத்தப்படும் தனிப்பட்ட தரவு (பயிற்சி, சரிபார்ப்பு, சோதனை என தரவு மூன்றாகப் பிரிக்கப்படும்). |
| Vector Database | திசையன் தரவுத்தளம் | திசையன் தரவு மையம் | திசையன் (vector) + தரவுத்தளம் (database). உட்பொதிவுகளை (Embeddings) சேமித்து ஒப்புமைத் தேடல் செய்யும் தரவுத்தளம்; RAG-ன் அடிப்படை. |
| Vector Embeddings | திசையன் உட்பொதிவுகள் | எண்வழி உட்பொதிவுகள் | திசையன் (vector) + உட்பொதிவுகள் (embeddings). சொற்கள், வாக்கியங்கள் அல்லது படங்களை AI புரிந்துகொள்ளும் வகையில், அவற்றின் பொருளைப் பிரதிபலிக்கும் எண்களாக மாற்றப்பட்ட வடிவங்கள். |
| Vector Search | திசையன் தேடல் | பொருள்சார் தேடல் | திசையன் (vector) + தேடல் (search). முக்கிய வார்த்தைகளை மட்டும் தேடாமல், பயனர் கேட்கும் கேள்வியின் ஆழமான பொருளைப் புரிந்து கொண்டு அதற்கு நெருக்கமான கருத்துகளைத் தேடிக் கண்டுபிடிக்கும் முறை. |
| Vibe Coding | சும்மா நிரலாக்கம் | உள்ளுணர்வு நிரலாக்கம் / உரைவழி நிரலாக்கம் / இயல்மொழி நிரலாக்கம் | கணினி மொழிகளை நேரடியாக எழுதாமல், மனித இயல்பு மொழியைக் கொண்டு AI மூலமாக ஒரு முழுமையான செயலியை உருவாக்கித் திருத்தும் முறை. ('சும்மா' = எளிதாக, இடையூறின்றி.) |
| Vibe Deploying | சும்மா வெளியீடு | உடனடி மென்பொருள் வெளியீடு | சும்மா நிரலாக்கத்தில் உருவாக்கப்பட்ட செயலியை, வழங்கி (Server) அமைப்புகள் இன்றி ஒரே தட்டலில் இணையத்தில் வெளியிடுதல். |
| Vision-Language Model (VLM) | பார்வை-மொழி மாதிரி | — | பார்வை (vision) + மொழி (language) + மாதிரி (model). படங்களையும் உரையையும் ஒரே நேரத்தில் புரிந்துகொண்டு செயல்படும் திறன் கொண்ட பன்முக மாதிரி. [^1] |
| Weight Decay | எடைத் தளர்வு | — | எடை (weight) + தளர்வு (decay). மிகைப்பொருத்தத்தைக் குறைக்கப் பயிற்சியின் போது எடைகளை மெதுவாகச் சிறிதாக்கும் சீரமைப்பு முறை. |
| Word Tokenization | தொடருடைத்தல் | சொற்பிரிப்பு | தொடர் (sequence) + உடைத்தல் (breaking). ஒரு வாக்கியத்தை அல்லது உரைப்பகுதியைத் தனித்தனி சொற்களாகவோ சொல்துண்டுகளாகவோ பிரிக்கும் இயன்மொழியாள்கைச் செயல்பாடு. [^1] |
| Word Vocabulary | சொல்லடைவு | சொல்லகராதி | சொல் (word) + அடைவு (vocabulary). ஒரு AI மாதிரி அடையாளம் காணக்கூடிய அல்லது பயன்படுத்தக்கூடிய ஒட்டுமொத்தச் சொற்களின் தொகுப்பு. [^1] |
| Zero-shot Learning | முன்மாதிரியற்ற கற்றல் | — | முன் (prior) + மாதிரி (example) + அற்ற (without) + கற்றல் (learning). முன்னதாக எடுத்துக்காட்டுகள் எதுவும் காட்டப்படாமலேயே, புதிய தரவுகளையோ பணிகளையோ மாதிரி தானாகவே புரிந்துகொண்டு செயல்படும் திறன். |
| Zero-shot Prompting | முன்மாதிரியற்ற தூண்டுதல் | சுழி-பயிற்சித் தூண்டுதல் | AI-க்குக் கட்டளை இடும்போது, எவ்வித எடுத்துக்காட்டுகளையும் உள்ளீட்டில் வழங்காமல் நேரடியாகக் கேள்வியைக் கேட்டுப் பதிலைப் பெறும் முறை. |

- **பதிப்பு 2:** முதல் கட்ட விரிவாக்கம்
- **பதிப்பு 3 (இந்தப் பதிப்பு):**
  - சொல்லாய்வு குழுவின் கலைச்சொற்களுக்கு முன்னுரிமை வழங்கப்பட்டது
  - சந்திப் பிழைகள் திருத்தப்பட்டன
  - **ஒற்றுச் சேர்ந்த சொல் விதி:** 2 சொற்கள் சேர்ந்தால் ஒற்று (மொழிமாதிரி, நரவலை); 3+ சொற்கள் சேர்ந்தால் "மாதிரி" தனியாக (பெருமொழி மாதிரி, பார்வை-மொழி மாதிரி)
  - **ஒருமை விதி:** அனைத்து ஆங்கிலத் தலைப்புகளும் ஒருமை வடிவில் — ANN, MLM, LAM, LCM, SLM, VLM, RNN, SAM, MoE Model (IoT விதிவிலக்கு — proper name)
  - வடசொற்கள் இயன்றவரை நீக்கப்பட்டன (பாரபட்சம், அங்கீகாரம், சமிக்ஞை, ஆரம்பம், நிஜம் — மாற்றப்பட்டன)
  - a + b + c சொல் வேர் விளக்கங்கள் முறையாகச் சேர்க்கப்பட்டன
  - 20+ புதிய Anthropic/OpenAI சூழல் கலைச்சொற்கள் சேர்க்கப்பட்டன (Reasoning Tokens, Tool Use, Tool Calling, System/User/Assistant Roles, Context Engineering, Mechanistic Interpretability, Constitutional Classifier, Sycophancy, Refusal, Steering, Activation Patching, Sparse Autoencoder, Circuit, Feature, Tokenization, Dataset, Latent Variable, Multi-task Learning, Sequence Modeling, Bias-Variance Tradeoff, Convolutional Neural Network, Generative Adversarial Network)
  - முக்கிய திருத்தங்கள்:
    - Abstraction: புறவயமாக்கம் → **சாரப்படுத்தல்**
    - Agent: செயற்பாட்டாலி → **செயல்முகவர்** (செயலியக்கி as alt)
    - AI Chatbot: **செய்யுரையினி** (சொல்லாய்வு) — primary, matches your article
    - Artificial Super Intelligence: செய்வியனறிவு → **செய்மீயறிவு**
    - Asynchronous: **ஒத்தியங்கா** (established term retained)
    - Synchronous: ஒன்னேர் → **ஒத்தியங்கு** (established term)
    - Critical Thinking Question: கிடுகறி வினா → **திறனாய்வு வினா**
    - Distillation: சுரக்காய்வு → **அறிவுச் சுருக்கம்**
    - Gradient: படிமச்சாய்வு → **சரிவு**
    - Gradient Descent: படிம இறக்கம் → **சரிவிறக்கம்**
    - Scalar: தன்மதிப்பு → **ஒற்றை மதிப்பு**
    - Self-Consistency: தன்முறை ஒருமை → **தன்னொத்திசைவு**
    - Sentiment Detection: சொற்சுவையறிதல் → **உணர்நிலை கண்டறிதல்**
    - Tensor: பன்மதிப்புக் கணம் → **பன்பரிமாண அணி**
    - Facial Expression Recognition: முகபாவ அங்கீகாரம் → **முகபாவம் அறிதல்** (வடசொல் நீக்கம்)
    - Bias: பாரபட்சம் → **சார்பு** (வடசொல் நீக்கம்)
    - System Prompt: ஆரம்ப அறிவுறுத்தல் → **தொடக்க அறிவுறுத்தல்** (வடசொல் நீக்கம்)
    - MoE Model: "வல்லுநர் கலவை மொழி மாதிரி" alternative removed (Language not in source term)
