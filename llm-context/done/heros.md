# Hero Images — Prompt Guide

## Master Prompt

Use this as the base for every chapter hero image. Append the chapter-specific section below it.

> **CRITICAL:** Do NOT include any Tamil script, Tamil letters (e.g. "அ", "தமிழ்"), or any text/words of any language rendered inside the image. LLMs generate Tamil text poorly and it will look broken. The image must be purely visual — no text overlays, no labels, no captions, no watermarks.

```text
Create a professional, highly conceptual editorial illustration for a Tamil AI glossary book chapter.

Style requirements:
- Editorial vector illustration style, minimalist and elegant, utilizing soft gradients, clean lines, and balanced negative space.
- Color palette: Muted, warm earth tones (terracotta, sandstone, ochre, soft beige) contrasted with a deep, rich green (#1a4d2e) as the primary focal accent. Soft, diffused lighting.
- 6×9 book-friendly aspect ratio.
- Strictly ZERO TEXT. No text, no labels, no letters, no words, no numbers, and no captions of any language anywhere in the image.
- Cultural integration: Seamlessly blend traditional Tamil cultural motifs (such as kolam geometric dot-grids, temple gopuram structural lines, or the elegant curves of a brass kuthuvilakku lamp) with modern technology and AI concepts. Use these motifs as subtle visual patterns, NEVER as text.
- Maintain a consistent, premium, and sophisticated art style across all chapters.
```

## Usage Instructions

1. Copy the **Master Prompt** block.
2. Append the desired **Chapter Prompt** block.
3. Feed to Gemini (or any high-quality image generation model like Midjourney/DALL-E 3).
4. Review: verify **zero text** appears in the output — regenerate if any text leaks through.
5. Save as `book/images/hero-ch{NN}.png` (e.g., `hero-ch00.png`, `hero-ch01.png`).
