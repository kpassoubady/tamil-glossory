# 99. வரைபட விளிம்பு வெண்கோடு சோதனை — Edge Casing Test

> **🎯 சோதனை நோக்கம்**
> - Mermaid வரைபடங்களின் இணைப்புக் கோடுகள் வெண்கோட்டுடன் (white casing) சரியாக வழங்கப்படுகின்றனவா எனச் சோதிக்க
> - ஒளி பயன்முறையில் (day mode) கருப்புக் கோடுகள் தெளிவாகவும், இருள் பயன்முறையில் (night mode) வெண்கோடு மூலம் தெரியவும் உறுதிசெய்ய
> - WeasyPrint வழியாக SVG சரியாக வழங்கப்படுகிறதா என PDF-ல் சரிபார்க்க

## எளிய வரைபடம் — Simple Flowchart

மூன்று உறுப்புகள் கொண்ட எளிய வரைபடம். இணைப்புக் கோடுகளும் அம்புகளும் கருப்பு உள்ளகத்துடன் வெண்விளிம்பு கொண்டவை.

![பேச்சுத் தொழில்நுட்பம்](diagrams/06-nlp-speech.svg)

## சிக்கலான வரைபடம் — Complex Pipeline

பல இணைப்புகள், துணைக்குழுக்கள், அம்புகள் கொண்ட வரைபடம். ஒவ்வொரு இணைப்பும் வெண்விளிம்பு பெற்றுள்ளதா எனப் பாருங்கள்.
